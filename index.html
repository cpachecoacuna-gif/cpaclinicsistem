🏥 Conectar Firebase

Solo tú haces esto una vez. Después todos abren la misma URL y ven todo en tiempo real.

Paso 1 — Crear proyecto
Ve a console.firebase.google.com → "Agregar proyecto" → nombre centro-medico → desactiva Google Analytics → Crear.
Paso 2 — Activar base de datos
Menú izquierdo → Realtime Database → "Crear base de datos" → ubicación us-central1 → selecciona Modo de prueba → Habilitar.

Icono ⚙️ → "Configuración del proyecto" → pestaña "General" → baja a "Tus apps" → clic en </> → Registra la app → copia los 4 valores que aparecen.
